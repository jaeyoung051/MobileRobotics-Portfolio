## 프로젝트 개요

본 프로젝트는 **AVR 기반 모바일 로봇 플랫폼**을 이용하여 로봇 미션을 수행하는  
모바일 로보틱스 대회 프로젝트입니다.

센서 데이터, 카메라 기반 객체 인식, 엔코더 기반 이동 제어를 이용하여  
로봇이 경기장 내에서 주어진 미션을 수행하도록 구현하였습니다.

프로그램은 **C 언어 기반 임베디드 환경**에서 개발되었으며  
ISP(In-System Programming)를 이용하여 로봇에 업로드하여 실행하였습니다.

## 수상 경력

- 2020 전국 기능경기대회 모바일 로보틱스 **은메달**
- 2020 지방 기능경기대회 **은메달**

## Tech Stack

- Language : C
- Microcontroller : AVR
- Development Tool : Atmel Studio
- Programming Method : ISP (In-System Programming)
- Sensors : IR Sensor, ID Sensor
- Vision : Camera-based Puck Detection
- Drive System : 3 Omni Wheel Holonomic Drive

---

## 로봇 시스템 구성

로봇은 **3 Omni Wheel 기반 Holonomic Drive 구조**로 제작되어  
제자리 회전 및 자유로운 방향 이동이 가능합니다.

사용된 주요 센서는 다음과 같습니다.

- IR 센서 기반 라인 트레이싱
- 금속 감지 센서를 이용한 경로 분기
- 카메라 기반 퍽(Puck) 인식
- 엔코더 기반 이동 거리 및 회전 제어

센서 데이터와 카메라 인식을 조합하여 로봇이 자율적으로 미션을 수행하도록 구현되었습니다.

---

## 담당 역할

본 프로젝트는 **2인 1조 팀 프로젝트**로 진행되었습니다.

제가 담당한 부분은 다음과 같습니다.

- 헤더 파일 작성 및 코드 구조 설계
- 동작 미션(1과제) 로직 구현
- 경기장 환경에 맞춘 맵(Map) 제작
- 카메라 기반 퍽 인식을 위한 ROI(Region Of Interest) 영역 설정
- 조명 환경에 따른 색상 인식 값 튜닝
- 로봇 동작 로직 수정
- 실제 경기 환경에서의 테스트 및 디버깅

팀원은 **경로 탐색 및 정렬 알고리즘(2,3과제)** 구현을 담당하였습니다.

---

## 경기 과제

대회는 총 3개의 과제로 구성되어 있으며,  
각 과제는 센서 데이터와 카메라 인식을 기반으로 로봇이 미션을 수행하도록 설계되었습니다.

### Task 1 – 동작 미션

1과제는 **카메라와 라인 센서를 기반으로 퍽(Puck)을 인식하고 처리하는 동작 과제**입니다.

로봇은 IR 센서를 이용한 라인 트레이싱을 통해 이동하며  
카메라를 이용하여 퍽을 인식한 후 조건에 맞는 경로를 선택하여 미션을 수행합니다.

- IR 센서를 이용한 라인 트레이싱
- 카메라 기반 퍽 인식
- 조건에 따른 경로 선택
- 미션 수행을 위한 로봇 동작 제어

---

### Task 2 & 3 – 정렬 미션

2과제와 3과제는 **경기장 내 원반 퍽을 지정된 위치에 정렬하는 과제**입니다.

이 과정에서 **길찾기 알고리즘(SMT 기반)**을 활용하여 로봇이 목표 위치로 이동하고  
컬렉터를 이용하여 퍽을 정렬하도록 구현되었습니다.

- IR / ID 센서를 이용한 위치 인식
- 라인 트레이싱 기반 이동
- SMT 기반 경로 탐색
- 컬렉터를 이용한 퍽 정렬

### SMT 기반 경로 탐색

2, 3과제에서는 경기장을 격자 맵으로 표현한 뒤,  
IR 센서와 카메라를 통해 실시간으로 맵 상태를 갱신하며 SMT 기반 경로 탐색을 수행하였습니다.

- 경로 단절 여부를 IR 센서로 확인하여 장애물 정보 반영
- 카메라를 통해 퍽 위치 및 종류를 인식하고 맵에 기록
- 방향 값(RR, UU, LL, DD)을 기반으로 이동 경로 계산
- 필요 시 대각 방향까지 고려하여 경로 유연성 확보

이를 통해 랜덤한 경기장 환경에서도 정렬 미션을 수행할 수 있도록 구현하였습니다.

### Competition Rules

대회는 **정확성과 속도를 동시에 평가**합니다.

- 로봇이 장애물이나 경기장 구조물에 충돌할 경우 감점
- 모든 퍽을 정확하게 정렬하면 기본 점수 획득
- 미션을 더 빠르게 완료할수록 추가 점수 획득

- 센서 데이터와 카메라 인식을 활용한 로봇 동작 제어
- 실제 로봇 테스트 및 디버깅 과정의 중요성
- 팀 프로젝트 환경에서의 협업 경험
